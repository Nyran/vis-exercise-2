Exercise 2: Scatterplot Matrix

Task 1: Cell Sizes

Compute the constants cellWidth and cellHeight which determine the size of each cell in the matrix. The final matrix should fill the width and height but leave the amount of pixels given in padding to the left and right of the matrix as well as between the rows and columns. Avoid non-integer values as they can lead to blurred lines in your final result.

Task 2: Color

Setup a color scale that encodes the categorical attribute species. If implemented correctly, the code cell below the declaration of color should display a legend for the color instead of the hint to implement the color scale. Choose an appropriate color scheme from the d3 color schemes.

Task 3: Scatterplot Matrix

The code cell below already sets up the scatterplot matrix but does not draw the data items. Implement the drawing of the scatterplot for each cell of the matrix. Use the scales for x and y that are created in the appendix of this document. See the lecture for an image of a scatterplot matrix of the same data to validate your result.

Task 4: Brushing and Linking

The function below sets up a brush for each cell of the scatterplot matrix and is used above. The d3 brush already provides all the brushing functionality. The brushing rectangle starts at (x0, y0) and goes to (x1, y1) with regard to the extent that was set for the brush. Implement the linking by de-emphasizing data items that are not inside the brushing area.
